src/components/ 폴더의 컴포넌트들:
Header.tsx: 웹사이트의 상단 헤더를 표시 (로고, 네비게이션 메뉴)
Navbar.tsx: 상단 네비게이션 바 (카테고리 메뉴, 검색, 장바구니)
Footer.tsx: 웹사이트 하단 푸터 (저작권, 링크)
BlogCard.tsx: 블로그 포스트 카드 형태의 미리보기
BlogList.tsx: 블로그 포스트 목록 표시
BlogDetail.tsx: 블로그 포스트 상세 내용 표시
ProductGrid.tsx: 상품 목록을 그리드 형태로 표시
ProductHero.tsx: 상품 페이지의 메인 히어로 섹션
설정 파일들:
package.json: 프로젝트 설정, 의존성 패키지 목록
vite.config.ts: Vite 빌드 도구 설정
tsconfig.json: TypeScript 설정
tailwind.config.js: Tailwind CSS 설정
postcss.config.js: PostCSS 설정
eslint.config.js: ESLint 코드 검사 설정
기타 파일들:
index.html: 웹사이트의 진입점
.gitignore: Git 버전 관리에서 제외할 파일 목록
dist/: 빌드된 파일들이 저장되는 폴더
